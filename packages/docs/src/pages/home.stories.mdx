## Princípios

No RDP, acreditamos em:

1. **Consistência**: Interfaces modernas garantindo experiências de usuário únicas.
2. **Acessibilidade**: Design para todos os usuários, considerando diversidades e limitações de usuários.
3. **Clareza**: Design claro, compreensível e amigável ao usuário.

## Components

### Formulários
 
 **Button**

- **Primary Button**: Use the primary color (#FF5733) with white text. 
- **Secondary Button**: Use the RDP Gray (#EAEAEA) with RDP Dark text (#333333).

### Visuais

**Avatar**
